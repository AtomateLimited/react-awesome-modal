# React-AwesomeModal

This is a React Component.

## Usage
```
<Modal
  visible={bool}
  closeFn={function}
>
  // some contents
</Modal>
```

## Setup
```
$ npm install
```

## Build
```
$ npm run build
```

## Watch and auto build
```
$ npm run watch
```

## Test
```
$ npm test
```
